# **App Name**: Encuentra la Palabra - Edición Bíblica

## Core Features:

- Configuración de Equipos: Configuración de equipos con nombres editables para fomentar la competencia.
- Desafíos por Niveles: Desafíos interactivos en dos niveles: 'Encuentra la Palabra' y 'Completa la Frase'.
- Diseño Responsivo: Diseño adaptativo para modos vertical y horizontal, optimizando la experiencia en diferentes dispositivos.
- Equipos Persistentes: Implementación de Firebase o almacenamiento local para guardar los nombres de los equipos de forma persistente.
- Sonido Integrado: Integración de efectos de sonido temáticos para mejorar la inmersión y la retroalimentación del juego.
- Estadísticas y Ranking: Uso de Firebase para guardar estadísticas, progreso y ranking de los jugadores.
- Monetización con AdMob: Implementación de AdMob con banners no intrusivos para la monetización.
- Localización: Textos en formato JSON para facilitar la traducción y localización a diferentes idiomas.
- Modo Silencioso: Opción para jugar sin sonido, ideal para entornos sensibles al ruido.
- Modo Práctica: Modo sin límite de tiempo, perfecto para el estudio bíblico y el aprendizaje.

## Style Guidelines:

- Color primario: Dorado profundo (#D4AF37), que recuerda al arte bíblico.
- Color de fondo: Beige claro (#F5F5DC), para evocar una sensación de antigüedad.
- Color de acento: Azul cielo (#87CEEB), que contrasta con el primario, sugiriendo revelación.
- Fuente para titulares: 'Playfair', serif. Fuente para el cuerpo del texto: 'PT Sans', sans-serif, que ayuda a mantener el texto principal fácil de leer. La primera le da un toque elegante y moderno a los encabezados, mientras que la segunda ayuda a que el texto del cuerpo sea accesible.
- Iconos estilizados inspirados en imágenes religiosas para logros y navegación.
- Transiciones y fundidos sutiles estilo pergamino para cambios de nivel y revelación de respuestas.
- Una vista de panel dividido en modo horizontal presenta preguntas junto con estadísticas en tiempo real, mejorando la participación del usuario.
