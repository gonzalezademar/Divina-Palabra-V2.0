# **App Name**: Divina Palabra - Edición Bíblica Juego Profesional

## Core Features:

- Configuración de Equipos: Configuración de equipos con nombres editables para fomentar la competencia de forma local.
- Desafíos por Niveles: Desafíos interactivos y expansivos en múltiples niveles y modos de juego.
- Diseño Responsivo: Diseño adaptativo para modos vertical y horizontal, optimizando la experiencia móvil y de tableta.
- Equipos Persistentes: Almacenamiento local seguro (`localStorage`) para guardar los nombres de los equipos y puntuaciones de forma persistente sin necesidad de servidores.
- Sonido Integrado: Integración de efectos de sonido temáticos con síntesis de audio (`Tone.js`) para mejorar la inmersión y la retroalimentación del juego.
- Aula Virtual (Lecciones Compartidas): Panel para educadores/líderes religiosos para crear lecciones temáticas, asignar niveles (Principiante, Regular, Avanzado) y formular preguntas de reflexión. Genera un enlace que puede ser compartido por WhatsApp, devolviendo la respuesta de reflexión del alumno al profesor por el mismo chat.
- Monetización Dual: Configuración para versión gratuita con anuncios integrados (AdMob) y opción de compra única integrada (In-App Purchase) para desbloquear la versión sin publicidad.
- Localización: Textos y consignas en formato JSON internacionalizado para facilitar la traducción de la interfaz a múltiples idiomas (Español, Inglés, etc.).
- Selector de Versiones de la Biblia: Selector al inicio que permite elegir jugar con textos adaptados dinámicamente a la Reina-Valera 1960 (RVR1960), Nueva Versión Internacional (NVI), u otras versiones destacadas según el idioma seleccionado.

## Style Guidelines (Premium Theme: "Pergamino Dorado y Cristal Oscuro"):

- Color de fondo: Slate profundo/Azul noche (#0f172a), sugiriendo misterio y solemnidad celestial.
- Color primario: Dorado profundo (#D4AF37), con efectos metálicos para títulos y bordes premium.
- Color de acento: Azul cielo (#87CEEB), para contrastes luminosos de respuestas y botones activos.
- Fuente para titulares: 'Playfair Display', serif elegante y moderna.
- Fuente para cuerpo: 'PT Sans', sans-serif, altamente legible en dispositivos móviles.
- Animaciones: Transiciones fluidas tipo paso de página de pergamino, efectos tridimensionales y de relieve en los botones (estilo neomorfismo de alta gama), y micro-animaciones al responder.
