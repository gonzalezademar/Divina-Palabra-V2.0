# Divina Palabra - Edición Bíblica

Un juego interactivo y educativo para poner a prueba y expandir tu conocimiento de las Escrituras. Desarrollado como una herramienta moderna para la formación, el estudio bíblico grupal y la evangelización.

## Características

- **Soporte Multilingüe:** Configuración inicial en múltiples idiomas.
- **Versiones de la Biblia:** Selección dinámica de versiones de la Biblia (Reina-Valera 1960, Nueva Versión Internacional, etc.).
- **Modos de Juego:** Retos de encontrar palabras, completar frases y adivinar la frase.
- **Aula Virtual:** Creación y distribución de lecciones personalizadas mediante enlaces estructurados compatibles con WhatsApp sin necesidad de servidores.
- **Diseño Premium:** Interfaz oscura con acentos dorados y relieves tridimensionales.

## Ejecución en Desarrollo

Para iniciar el servidor de desarrollo local:

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver la aplicación.
